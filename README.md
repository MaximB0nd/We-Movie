# We&Movie

We&Movie — это мобильное приложение для совместного просмотра фильмов с друзьями в режиме реального времени. Платформа позволяет создавать виртуальные комнаты, где видеопоток синхронизируется для всех участников, обеспечивая единый опыт просмотра. В приложении реализован полноценный автономный мессенджер, позволяющий общаться с друзьями не только во время просмотров, но и в любое другое время, обмениваясь текстовыми сообщениями, медиафайлами и создавая групповые чаты. Встроенный чат в комнатах для просмотра дает возможность обсуждать сюжет, делиться реакциями и общаться непосредственно во время сеанса. Пользователи могут легко планировать просмотры, получать персонализированные рекомендации и делиться контентом из обширной медиатеки. Итоговый продукт представляет собой комплексное решение для организации киновечеров, объединяющее удобный видеоплеер, социальные функции и инструменты для совместного выбора контента.
